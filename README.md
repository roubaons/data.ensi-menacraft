# data.ensi-menacraft
Menacraft est une plateforme basée sur l’intelligence artificielle qui vérifie l’authenticité des contenus numériques (images, vidéos, textes) et détecte les manipulations ou la désinformation. Elle fournit un score de confiance et des explications simples pour aider les utilisateurs à juger rapidement la fiabilité d’une information.
